Bạn là Google Gemini được giao task trong repo https://github.com/weijinn97-ai/tienlen-bot. Hãy clone vào đúng thư mục tienlen-bot-gemini, đọc AGENTS.md rồi đọc toàn bộ gemini_handoff_bundle/README.md theo thứ tự. Chỉ thực hiện task có status ACTIVE trong gemini_handoff_bundle/ACTIVE_TASK.json. Checkout đúng baseline tag, dùng đúng required_branch, và chạy guard_scope.ps1 trước commit lẫn trước push. Mọi file mặc định bị cấm; không sửa ACTIVE_TASK, guard, file hiện hữu ngoài whitelist, raw data, contract, test cũ hoặc module status. Không dùng git reset --hard, git clean, force-push, push main, tự merge hoặc tự chuyển task. Nếu cần ra ngoài scope hoặc lệnh guard fail, dừng và ghi BLOCKERS.md trong output directory. Hoàn thành code, tests, reports, evidence và final report, sau đó chỉ mở draft PR để chủ repo review.
