Current reality

Du an dang o dau va con thieu gi

Da co va dung duoc
  • Scan va bind dung gia lap MEmu theo vm_index, title, pid, hwnd, adb_serial.
  • Launcher chon instance, start, stop, xem log realtime.
  • Watchdog, validation co ban, multi-frame consensus scaffold.
  • ADB broker, supervisor, worker va inference scaffold.
  • Google Sheet tracker va bo seed CSV da duoc cap nhat.
  • Image intake workflow cho nhieu agent va HTML handoff bundle.
Chua xong truoc khi coi la bot choi that
  • Capture backend production va live preview trong launcher.
  • Perception YOLO/OCR that tu frame game.
  • State extraction that tu man hinh game.
  • Action execution that va verify sau thao tac.
  • Decision engine Tien Len theo luat/chien luoc that.
  • Multi-bot supervision full va resource control that.
Thu tu khuyen nghi de lam tiep: graceful stop → live preview capture → perception pipeline → state extraction → action execution → post-action verify → provider-specific API adapter.
Architecture guardrails

Nhung nguyen tac phai giu

Identity
  • Mot bot gan voi mot worker/process rieng.
  • Khong map bot theo vi tri cua so hay thu tu window.
  • Dinh danh chinh la title, vm_index, pid, hwnd, adb_serial.
Runtime
  • Capture la Windows-side; khong dua adb screencap vao hot path.
  • Frame queue bounded va latest-only.
  • Neu state chua chac, khong hanh dong mu.
Recovery
  • ADB di qua broker hoac lop dieu khien da chuan hoa.
  • Validation va consensus quan trong hon FPS.
  • Bot loi thi pause/restart rieng le, khong reset ca cum.
